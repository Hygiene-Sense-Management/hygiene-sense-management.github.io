HsM OEM Customers

 Top  Previous  Next

Entities in the Catalog Service are mapped to one or more OEM Customers. E.g. a device type is mapped to one or more OEM-Customers.

When Accessing the Catalog Service’s admin API all read and write operations will verify if the accessed or modified entity is mapped to an OEM customer that is also in the list of OEM-Customers administrated by the accessing user or client.

As the entities managed by the Catalog Service do not contain any business secrets and therefore the Browse API can be accessed anonymously, the Catalog Service’s browse API also does not protect entities with regard to the OEM-Customers they belong to. However for accessing entities through the browse API the user or client must know the entity’s ID, or the OEM-Customer ID.
