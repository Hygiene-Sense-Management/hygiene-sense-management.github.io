Filters for devices that are of one of the given device types.

Settings

Setting

Description

Device types

A list of device types (as defined in the Catalog Service). The result set will only include devices whose device type matches with one of the given device types.

If the list is empty the result set will be empty (as no device type matches)

Validation Rules

•Device types list must contain at least one device type: If no device type is given this term will not match any device.

•All device types must be valid device types, i.e. must be device types registered in the Catalog Service. If the given device type id is not valid the results will not list any devices of that non-existing type.
