Location of configuration file

 Top 

All  configuration for Digital Twin Registration Agent is done through a configuration file.

Which configuration file is used depends on the --Environment command line switch passed to the Digital Twin Registration Agent and the system environment variable DTRA_CONFIG_DIR. The actual configuration file used is then. <DTRA_CONFIG_DIR>\appsettings.<environment>.json

If the configuration file is not found the Digital Twin Registration Agent will not start, this exception is logged to the windows event log (when started as a service) or to the terminal.
