Location tree filter

 Top  Previous  Next

Filters for devices that are installed at specific locations with in the location tree of customers.

Settings

Setting

Description

Locations

Devices that are assigned any of the given locations or any of its sub-locations will pass the filter.

If the list is empty the result set will be empty (as no device type matches)

Validation

•At least one location must be set: If no location is set this filter term will not match any device.

•All locations must be valid locations
