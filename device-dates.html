Installation date

 Top  Previous  Next

Filters for devices based on their installation date, i.e. the date when the device was installed at a specific locaiton.

Settings

Setting

Description

Installation type

•First installation: Date of first installation of this device in one of the customers locations.

•Current installation: Date of installation of a device at the current customer

Date interval

An adaptive or static date interval

 

Validation rules

•Date interval must be a valid date interval and must include past dates.
