Compound device status

 Top  Previous  Next

Compound devices are part of the base set of devices to which device filters are applied. The Combined device filter term can be used to restrict the result set to non-combined devices.

Settings

Setting

Description

Compound status

A single selection of one of the following options:

•Is compound member device: Devices that are linked to other devices

•Is compound device: Devices that include other devices

•Is not compound device

•Is not compound member device

Validation rules

None
