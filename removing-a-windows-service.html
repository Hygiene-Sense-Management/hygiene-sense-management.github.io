Removing a Windows Service

 Top 

Using PowerShell

Note you need to run PowerShell with administrative privileges to modify Windows Services.

Removing a Windows Service for the Digital Twin Registration Agent can be done with the Remove-Service PowerShell cmdlet. For full documentation of this cmdlet type Get-Help Remove-Service -Online  in PowerShell. Note that this cmdlet was introduced with PowerShell 6.0 and is not available in previous versions.

General Syntax of the Remove-Service cmdlet:

Remove-Service 

 [-Name] <String> 

 [-WhatIf] 

 [-Confirm] 

 [<CommonParameters>]

 

The -Name parameter must match the -Name parameter used when creating the Windows Service. Example

Remove-Service -Name DTRA_Production

Using sc.exe

As alternative to the PowerShell cmdlet you can use the sc.exe tool

General Syntax of the Remove-Service cmdlet:

sc.exe [<servername>] delete [<servicename>]
