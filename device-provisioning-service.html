Device Provisioning Service

 Top  Previous  Next

HsM uses an Azure Device Provisioning service for Stationary gateways so that these gateways will initially connect to the right HsM OeM Partner's HsM Backend infrastructure.

After production each stationary gateway is registered with the device provisioning service. The registration record links the gateway to a specific Azure IoT hub and it provides the keys for the device to be able to communicate with the provisioning service.
