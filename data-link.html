Data link capability

 Top  Previous  Next

The HsM system can also manage pure analog dispensers (with no data link to the HsM HsM Backend), this device filter term type can be used to filter devices by their capability to send data to the HsM <%BACKEND%.

Settings

Setting

Description

Capability filter

Yes: Filters for devices that can communicate with the HsM backend.

No: Filters for devices that cannot communicate with the HsM backend.

Compound member handling

Include Devices with compound members that have data link capability

Validation rules

None
