Environment variable DTRA_CONFIG_DIR

 Top 

The Windows System Environment Variable DTRA_CONFIG_DIR defines the directory where configuration files are expected and where the Digital Twin Registration Agent will create subfolders for storing data. Note that the environment variable must be a System environment variable.

The DTRA_CONFIG_DIR variable should point to a local directory and that directory should be included in your backup.You can use the Windows shell to set the environment variable. Example:

setx /M DTRA_CONFIG_DIR "D:Data\Hsm\Digital Twin Registration Agent"

If this system environment variable is not set the Digital Twin Registration Agent will not start, also if the Digital Twin Registration Agent does not find a configuration file that fits the Environment setting
