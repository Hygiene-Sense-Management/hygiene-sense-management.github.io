Display Groups

 Top  Previous  Next

The display groups defined with a firmware variant version are a partition of the set of resource instances defined with that firmware variant version (i.e. form mutually disjoint sets of resource instances, and each resource instance is assigned to a display group.)

Display groups give clients of the catalog service a hint how they should visualize the resource instances of a firmware.

 

One firmware has a single 1-level tree structure in which it can organize resource instances

Clients can use this structure to show resource instances in a tree-like structure.

Clients can additionally filter by:

•Read | Read-write | write-only

•Visibility
