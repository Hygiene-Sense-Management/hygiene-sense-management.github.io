Updating

 Top 

Updating the Digital Twin Registration Agent is done by replacing all the files int he program directory with the files of the new version - just make sure you exit all running instances of the Digital Twin Registration Agent before first.

As the Digital Twin Registration Agent does not keep any configuration information in the program directory (unless, and against our advice, you really decided to do so in your configuration) updating the Digital Twin Registration Agent does not destroy any data.

Note: The version the Digital Twin Registration Agent is stored in the .exe PE file header and is therefore shown in Windows Explorer's file properties dialog.
